# Extending Lox

These are the documented changes to the language/syntax from the original Lox language.

## Next: 0.2.0

- Short forms of `and`/`or`: `&&` and `||`
- Nullish coalescing operator (`??`)

## 0.1.0

- No class inheritance
- Native functions added
  - `exit`
  - `__version__`
  - `rand`
  - `rand01`
  - `randBetween`
  - `ceil`
  - `readFileToString`
  - `writeStringToFile`
  - `fileExists`
  - `getenv`
  - `setenv`
  - `len`
  - `typeof`
  - `argc`
  - `argv`
  - `prompt`
  - `stdin`
  - `instanceOf`
  - `parseNumber`
