# CLI

## Language Version

```sh
clox -v
clox --version
```

## REPL

```sh
clox -r
clox --repl
```

## Run File

```sh
clox -f path/to/file.lox
clox --file path/to/file.lox
```

## Run Code

```sh
clox -c 'print "Hello, World!";'
clox --code 'print "Hello, World!";'
```

## Help

```sh
clox -h
clox --help
```
