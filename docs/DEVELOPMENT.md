# Development

## Important Files

- [justfile](../justfile) Build, run tests, etc... See: https://just.systems/man/en/
- [src/main.c](../src/main.c) REPL, Run \*.lox files
- [src/compiler.c](../src/compiler.c)
- [src/vm.c](../src/vm.c)
- [src/memory.c](../src/memory.c)
- [VERSION.txt](../VERSION.txt) Defines the version of the language. See [Incrementing Language Version](../scripts/README.md#increment-language-version) for changing this.
- [scripts/tests.sh](../scripts/tests.sh) Runs the language tests (E2E, snapshots)
- [scripts/coverage.sh](../scripts/coverage.sh) Generate coverage information
- `src/version.c` Defines `CLOX_VERSION` from `VERSION.txt`. Generated by [`scripts/generate_version_c.sh`](../scripts/generate_version_c.sh) when the project builds. This file is not tracked in source control.

## Adding Functionality

### Create A Native Function

- [ ] 1. Declare a new function in [vm.c](../src/vm.c)
- [ ] 2. Register the native function with `defineNative("function_name", functionNameNative)` in `initVM`
- [ ] 3. Add [tests](../tests/README.md) for the new function
- [ ] 4. Add function to [`clox.tmLanguage.json`](../vsc/syntaxes/clox.tmLanguage.json) under `"name": "entity.name.function.native.clox"`
  - [ ] Call
  - [ ] Call with no args (if applicable)
  - [ ] Call with too many args (if applicable)
  - [ ] Call with incorrect arg types (if applicable)
